# Sistema-de-contagem-de-visitas
Sistema de Registro de Visitas desenvolvido com HTML, CSS, JavaScript, AJAX, PHP e MySQL, com cadastro de visitantes, controle de acessos, histórico, painel administrativo, filtros e relatórios.

# 📝 Sistema de Registro de Visitas

Sistema Full Stack desenvolvido para realizar o controle de entrada e saída de visitantes em empresas, condomínios, instituições e ambientes corporativos, proporcionando maior segurança, organização e rastreabilidade das visitas.

---

# 📖 Sobre o Projeto

O Sistema de Registro de Visitas foi desenvolvido com o objetivo de digitalizar e automatizar o processo de controle de visitantes, substituindo registros manuais por uma plataforma moderna, intuitiva e eficiente.

A aplicação permite cadastrar visitantes, registrar horários de entrada e saída, acompanhar o histórico de visitas e gerar relatórios para facilitar a gestão e auditoria das informações.

---

# 🚀 Funcionalidades

- Cadastro de visitantes
- Registro de entrada e saída
- Controle de horários
- Histórico completo de visitas
- Pesquisa por nome, documento e data
- Painel administrativo
- Controle de usuários e permissões
- Estatísticas de visitas
- Exportação de dados (CSV)
- Edição e exclusão de registros
- Comunicação assíncrona utilizando AJAX
- API em PHP com respostas em JSON
- Estrutura preparada para futuras integrações

# ⚙️ Como Executar

## 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/sistema-registro-visitas.git
```

## 2. Configure o servidor

Utilize um servidor local como:

- XAMPP
- WAMP
- Laragon

## 3. Importe o banco de dados

Abra o phpMyAdmin e importe o arquivo SQL disponível na pasta do projeto.

## 4. Configure a conexão

Edite o arquivo responsável pela conexão com o banco de dados informando:

- Host
- Usuário
- Senha
- Banco de Dados

## 5. Execute o projeto

```
http://localhost/
```

# 🏗️ Arquitetura

O projeto segue uma arquitetura Cliente-Servidor.

```
Usuário
    │
    ▼
HTML + CSS + JavaScript
    │
    ▼
AJAX
    │
    ▼
PHP
    │
    ▼
MySQL
```

Toda comunicação entre Front-end e Back-end ocorre através de requisições AJAX utilizando JSON como formato de troca de informações.
